Int-Far is an interactive Discord bot that tracks various games being played in Discord servers.

Since 2020-07-09 Int-Far has tracked {{ games }} games played by {{ players }} people in {{ active_guilds }} Discord servers.

Int-Far works by first linking a user's Discord ID and in-game ID (summoner ID for LoL, Steam ID for CS2). Then, it tracks the games of people in Discord voice channels.

When a game is over, a summary is written, stats are saved, and awards for being really bad (somneone is Int-Far) or really good (someone got doinks) are handed out.
